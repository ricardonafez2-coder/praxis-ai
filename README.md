# Praxis AI Platform 🚀

Plataforma de Simulación de Pacientes Virtuales con IA para instituciones educativas con carreras de salud (Psicología, Medicina, Odontología, Fisioterapia).

**Stack:** Next.js 16 + ElevenLabs + Simli + Supabase  
**Deploy:** https://praxis-ia.netlify.app

## Getting Started

First, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser.

## Environment Variables

Create a `.env.local` file with:

```
SIMLI_API_KEY=
ELEVENLABS_API_KEY=
ELEVENLABS_AGENT_FATIMA=
NEXT_PUBLIC_SUPABASE_URL=
NEXT_PUBLIC_SUPABASE_ANON_KEY=
SUPABASE_SERVICE_ROLE_KEY=
NEXT_PUBLIC_APP_URL=
```
